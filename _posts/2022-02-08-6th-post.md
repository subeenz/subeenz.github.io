# **Streamlit** 시작

모델 배포를 위한 프론트엔드 환경을 구성하기 위해 Streamlit 공부를 시작하겠습니다.

사용할 수 있는 데코레이터는 https://docs.streamlit.io/library/get-started 을 통해 확인할 수 있습니다.

각 데코레이터의 사용방법을 보기 쉽게 정리해두려고 합니다.

```python
import streamlit as st
```

실행방법은 다음과 같습니다.
```bash
$ streamlit run [file_name]
```

```python
@st.write()
```

```python
@st.title()
```

```python
@st.head()
```

(캐싱 : 성능을 위해 메모리 등에 저장하는 행위)
매번 다시 실행하는 특성 때문에 데이터도 매번 다시 읽을 수 있음
모델을 로드하는 경우 모델의 크기가 크면 로드하는 시간이 매번 오래걸리기 때문에

```python
@st.cache
```
